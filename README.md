## ğŸ—„ï¸ Database Structure

The database consists of **3 core tables**:

- **Books**  
  Stores book details such as title, author, genre, price, published year, and stock quantity.

- **Customers**  
  Contains customer information including name, email, phone number, city, and country.

- **Orders**  
  Records customer purchases by linking books and customers, along with order date, quantity, and total amount.

This simplified structure is designed to represent a single-book purchase per order for learning and analysis purposes.
